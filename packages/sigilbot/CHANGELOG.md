# Changelog

All notable changes to `@heysigil/sigilbot` are documented in this file.

## [0.1.0] - 2026-02-19

### Added

- Initial deployable Sigil bot stack package.
- Runtime endpoints for health, capabilities, handshake, chat passthrough, and bot listing.
- MCP token scope validation during bootstrap and handshake.
- Optional `sigil-mcp` sidecar boot support.
- Dockerfile, Railway config, and environment template for one-click style deployment.
